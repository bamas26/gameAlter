# 앨리스 크로니클
## 1. 게임의 목적
- 배경 세계관
  - 현재 학원도시 바스크 각 지역에는 군사 학교들이 자리 잡고 있었다. 
  - 군사 학교의 각 구성원들 즉, 학생이라 불리는 이들을 앨리스라 통칭했는데
  - 태생이 난폭하고 금강불괴의 앨리스들은 편을 먹고 시가전을 벌이거나 대규모 전투를 벌이기를 좋아해
  - 바스크는 항상 난장판이었다.
  - 그러나 신력 2012년 도시의 안정을 위해 베르샤르, 엘사르, 아르다음사르 세 군사학교의 동맹이 이루어진 후
  - 바스크는 급속도로 안정되었고 평화의 시대가 도래 하였다.  

- 사건의 발단
  - 신력 2023년 우리 도시에 새로운 학교가 생겼다.
  - 놈들은 징그러운 얼굴과 이상한 복장을 하고 바스크의 질서를 어지럽히기 시작했다.
  - 시가전이 일상으로 벌어지기 시작했고 점점 피해를 입는 친구들이 나타났다.  

- 목적
  - 동맹은 이를 지엽적인 문제로 해석하고 적극적으로 나서지 않았다.
  - 도시 전체를 아우르는 관리기구가 없는 바스크에서 결국 녀석들에게 피해를 받던 친구들이 하나둘 모여 세력을 규합한다.
  - 각각의 학교를 중심으로 반격이 시작 됐다.  

## 2. 게임 시스템
1) 전투        
  - 전투 방식
    - 획득한 동료를 전장에 최대 5명까지 배치하여 전투
      - 전장의 병사들은 스킬 사용 (화면에 표시되지 않는다.)
      - 중단에 리더의 스킬이 오픈
        - 리젠후 사용 가능  
    - 웨이브중의 적의 체력을 빼앗으면 승리 (서로 죽지 않는 금강불괴의 몸이지만 기절, 넉백은 가능)
      - 웨이브는 4구간으로 구성 각 구간마다 일정수의 몹이 나타나고 마지막 구간에 보스 등장
      - 보스를 클리어 하면 레벨 상승
         
2) 성장
  - 보급
    - 골드를 지급하면 후방에서 보급품이 도착한다. 전투 부대의 명성과 중요도에 따라 우선 배급되기 때문에 보급의 시간이 줄어든다.
    - 상단 슬롯 (공격력 증가) - 모자, 장갑,신발
    - 중단 슬롯 (체력 증가) - 배지, 가방, 헤어핀
    - 하단 슬롯 (특수) - 목걸이(치유 증가), 부적(저항 증가), 손목시계(치명 증)     
    - 우측 고급 스롯 - 무기 (공격력 증가)   
  - 소환
    - 동료와 스킬을 소환
    - 같은 등급의 동료나 스킬이 나오면 수량으로 강화
    
3) 재화 수급
  - 보석
    - 수급처 : 퀘스트(메인, 일일) 완료시, 보스 던전 클리어시, 융합로에서 수급
    - 사용처 : 뽑기 (스킬, 동료), 즉시완료처
  - 골드
    - 수급처 : 전투에서, 골드 던전클리어시, 융합로에서 수급    

## 3. 컨텐츠
1) 메뉴
  - 보급 : 기본 슬롯 / 고급 슬롯
    - 기본 슬롯은 3개이고 각각 3가지 아이템으로 총 9개 아이템
    - 고급 슬롯은 무기  
  - 캐릭터 : 승급 / 스킬 / 유물
    - 전직 : 계급이 상  
  - 동료 : 보유 / 강화 / 장착 
  - 던전 : 보스 / 골드 / 장비 / 명성 / 목걸이 / 반지 
  - 사원 : 아슬란의 신을 믿는 사원으로 아슬란의 누구도 신앙심이 깊다.
    - 부활 / 해독 / 치료 -> 전투중 일어난 일들을 시간으로 치료 (보석으로 즉시 완료)
    - 개간 : 신성한 노동을 해야 하는 의무
    - 훈련 : 군인으로써 훈련을 통해 스탯 강화 (시간 소모 - 보석으로 즉시 완료)
    - 경전 연구 : 적을 새겨 놓는다.  
  - 상점 : 소환 / 패키지 / 스페셜 / 재화

2) 병사
  - 직업
    - 탱커 : 높은 체력과 회피로 최전방에서 적의 공격을 유도 반드시 필요한 직업
    - 딜러 : 우수한 공격력으로 적의 체력을 깎는 병과 (팀은 주로 이 병과로 구성)
    - 서포터 : 다양한 버프/비더브플 뿌리는 병과
    - 힐러 : 치유 병과 탱커와 함께 반드시 필요
    - T.S : 지원병과로 팀에 등장하지 않는다. -> 리더 스킬을 통해 등장하는 비행기, 비행선, 탱크등.. 
  - 배치
    - 전열 - 샷건, 기관단총, 화염방사기및 탱커는 모두 여기에 속한다.
    - 중견 - ㅜ건총, 돌격소총, 유탄발사기
    - 후열 - 기관총, 저격소총
  - 공격과 방어타입
    - 주로 인간대 인간으로 전투가 이루어짐
    - 방어는 특별한것이 없고 공격은 일반, 폭팔, 관통, 신비, 진동의 타입이 있다.
      - 일반 -> 파급효과가 없고 최초 1인에게만 데미지를 준다.
      - 폭팔 -> 범위에 따라 파급효과가 있다. (피해감소는 캐릭터의 능력에 따라)
      - 광통 -> 최초 피격자의 후면에 피해 (능력에 따라 사람수로 결정)
      - 오라 -> 마치 독처럼 시간이 흐를때마다 피가 빠진다. (자연 치료 불가)
      - 암흑 -> 시체처럼 멈춤   
  - 리더와 팀원
    - 리더는 스킬을 사용하여 팀원을 독려한다.
      - 리더스킬
        - 집중 사격 -> 리젠이 70% 이상인 팀원은 스킬 시전
        - 공중 지원 -> 비행기에서 폭탄 투하
    - 팀원은 동료로 참가
      - 기본 스킬, 고유 스킬, 궁극 스킬을 사용한다.
        - 기본 스킬 -> 공속을 기준으로 하는 기본 공격
        - 고유 스킬 -> 연속 발사, 조준 발사등 동료마다 특정된 스킬
        - 궁극 스킬 -> 매달 1회씩 오픈되는 동료 이벤트를 완료하면 얻어지는 스킬

## 4. BM\
1) 광고
- 보상 광고
  - 능력 상승 -> 광고보고 축복 (공격력, 골드, 스킬 시전시간 각각 25% 상승)
  - 보상 2배 -> 방치 보상, 데일리 보상, 레벨 보상
  - 뽑기 -> 동료와 스킬 소환시 일일 2회 광고 보고 무료 뽑기
- 사이 광고 없음
  - 광고 제거를 목적으로 하는 사이광고는 없음 
2) 패키지 판매
- 병사를 중심으로 하는 패키지 구성
  - 주로 병사 + 던전 열쇠 + 개간 관련 아이템이 주요 구성
  - 혹은 상급 병사 + 하급 병사 다수의 구조 (상급은 현재 뽑기 레벨에서 상위 확률 병사) 
3) 패쓰 판매
- 시즌 패쓰, 스페셜 패쓰 판매
  - 보상 차별화를 목적으로 월정액 패쓰 판매 
4) 재화 판매
- 보석 판매
  - 뽑기가 주 목적인 보석 판매
- 골드 판매 없음

 
# 삼국무장소녀
## 1. 게임의 목적
- 개요
  - 삼국지의 무장들을 모두 여성을 표현한 방치형 게임
  - 관우, 조운, 황충등의 무장과 순유, 진궁, 곽가등의 법사, 동탁등의 군벌들이 주축이 된다. 
- 목적
  - 후한말 어지러운 난세는 평정하여 통일 왕조를 세우는 것
    - 지역마다 점령하여 해당 세력을 규합하여 군사력을 강화한다.
    - 역사와 달리 무장을 마음대로 선택하여 새로운 세력을 만들수 있다.    

## 2. 게임 시스템
- 강화
  - 기본 -> 공격력, 체력, 체력회복, 크리티컬 피해, 크리티컬 확률, 더블 타격, 트리플 타격, 공격 스피드
  - 추가 -> 일반 공격력 추가, 고급 공격력 추가  
- 플레이어
  - 조조, 유비, 손권등 셋중 하나를 선택하여 게임을 시작
  - 유저가 선택한 무장을 주장이라 하고 독자적인 이름을 가질수 있고
  - 장비를 강화하며 스킬을 사용할수 있다.
  - 장비
    - 무기와 방어구는 뽑기로 
    - 목걸이와 반지는 던전을 통해 얻는다.
  - 스킬
    - 리더 스킬로 집중 공격을 선택하면 동료가 특정 공격을 한다거나 하는 따위의 스킬
    - 투석기 공격 스킬도 존재
    - 각각 스킬의 호용은 조조, 유비, 손권마다 다르다.
  - 유물
    - 선조의 유물을 확률로 얻으면 스탯이 강화된다.      
- 동료
  - 당대에 나온 삼국지의 인물들이 동료로 출현한다.
  - 각각의 장수는 빨파녹의 칼라로 속성이 구분되며 각각 위촉오를 대변한다.
    - 조조는 위와 유비는 촉, 손권은 오와 120% 효율을 나타내고
    - 조조와 촉, 유비와 위의 무장은 모두 75% 효율을 나타낸다.
    - 오의 무장은 각각 다른 성향을 가진다. (개별적)   
  - 휘화에 두면 보유효과를 얻을수 있고 장착하면 전장에 나간다.
  - 동료는 모두 기본, 고유, 궁극 스킬을 사용하고 이중 궁금 스킬은 개별 무장 이벤트를 해야 얻울수 있다.
- 던전
  - 쌀을 얻는 보스 던전, 금화를 얻는 보급, 장비를 얻는 ?, 명성을 얻는 공성
  - 반지와 목걸이를 생성하는 던전    
- 훈련과 개간
  - 훈련
    - 스탯을 강화하는 훈련 -> 시간을 소모한다 -> 쌀로 즉시 완료
  - 연구
    - 개간및 훈련 관련 연구  
  - 개간
    - 황무지를 개간하여 재화를 얻는다.
      - 곡궹이 -> 한칸 한칸 개간
      - 소 -> 특정 범위를 개간
      - 일꾼 -> 힘좋은 일꾼이 빠르게 일을 한다.
     
- 상점
  - 소환 -> 장비, 스킬, 동료
  - 패키지 -> 무장을 중심으로 패키지를 구성
  - 스페셜 -> 광고 제거, 자동 축복, 프리미엄 패쓰
  - 재화 -> 다이아 구매

## 3. BM
- 광고
  - 보상 광고 -> 축복, 보상 2배, 광고뽑기
- 패키지
  - 무장을 중심으로 황소나 던전 입장 열쇠를 묶어서 패키지 구성
  - 재화를 중심으로 열쇠와 묶어서 패키지 구성
  - 무장 세트 -> 고급 무장 세트   
- 스페셜
  - 자동 축복
  - 패쓰 -> 보상 차별화   
- 재화 구매
  - 골드는 팔지 않는다. 
 
# 몬스터 
## 1. 물질계
- 골렘
  - 물리공격위주. 체력과 공격력이 강하다 체력회복은 되지 않는다. 마법 공격이 통하지 않는다. 
  - 스톤 골렘
  - 골드 맨 - 금으로 된 골렘 -> 체력회복이 가능하다.

- 킬러 아머
  - 몸은 없이 갑옷으로 돌아다닌다. 물리 공격만 하는 골렘과 비슷하지만 2회 공격을 하는 몹, 같은 이유로 마법과 상관없다.
  - 킬러아머 보우
  - 킬러아머 나이트
  - 방황하는 갑옷 -> 무기가 없는 몬스터

- 박스 몬스터
  - 박스와 동일하게 생겼으나 건드리면 몬스터로 변한다.
  - 트랩 몬스터로 허접한 캐릭이나 강화하면 나중에는 아주 강력한 폭팔력을 가지게 된다.  

- 블리쟈드맨
  - 온몸에 전기를 가지고 있는 몬스터로 빠른 시간에 체력이 회복되는 능력을 가졌다.
  - 공격력이 압도적이어야만 이겨낼수 있다.
