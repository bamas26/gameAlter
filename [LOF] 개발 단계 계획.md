# 개요
- 쟝르는 기본적으로 방치혀 게임으로 설정하고 시작한다.
- 다만 해당 게임의 소재나 방치형 게임에도 다양한 메커니즘이 있기 때문에
- 이를 합리적인 의사결정을 통해 진행하기 위해 아래의 프로세스틀 통해 결정한다.
    
    - <b>제작</b> : 게임의 컨셉과 핵심 매커니즘이 담긴 게임 프로토타입 제작 
    - 크리에이티브 : 게임 컨셉과 플레이가 담긴 영상 크리에이티브 제작 (약 3-4종 베리에이션) 
    - 마케팅 : 목표 예산 설정 및 광고 세팅 (일반적으로 첫번째 테스트에서는 Facebook 플랫폼 활용) 
    - 사용자 : 타겟된 사용자들이 해당 광고를 보고 게임 다운로드 & 플레이 
    - 분석 : 해당 기간의 CPI 확인 및 기본 게임지표 분석

# 1. 프로토 타입 (CPI TEST 준비) 
## CPI 테스트 1단계
- 소재 선정 단계 (제작 기간 1달)종의 
- 전투엔진은 횡스크롤 진행 방식으로 일반적인 방치형 게임과 동일하다.
- 전투는 동일하고 그래픽 소재만 다르게 설정
- CPI 테스트를 통해 세가지 타입중 하나를 선정한다.

1) 기본 타입
    - 물고기가 소재
    - 리버스 빌런 세계관
    - 자동 사냥이 기반이된 전투 시스템
    - ex) 레전드 오브 슬라임류

2) 미소녀 타입
    - 미소녀가 소재
    - 삼국지의 무장 -> 미소녀 세계관으로
    - 자동 사냥이 기반이된 전투 시스템
    - ex) 슬레이어 키우기 류

3) B급 정서 타입
    - 외계인 혹은 몬스터가 주인공
    - 우주?
    - 자동 사냥이 기반이 된 전투 시스템
    - ex) 피클페이트
 
## CPI 테스트 2단계
- 전투엔진 선정 단계 (제작 기간 1달)
- 전투 엔진의 방식을 결정한다.

1) 기본 타입
    - 기본 타입
    - 주인공의 기본 공격이 있고
    - 우측에 나타나는 적을 물리치며 전진
    - 타입별
        - 물고기류 -> 입에서 돌을 던진다.
        - 미소녀류 -> 무기를 휘둘러 공격
        - B급 류 -> 총을 사용 

2) 캣 히어로 타입
    - 기본 공격 무기를 합쳐서 공격력을 상승 시킨다.
    - 유저가 시간을 들여 노력하면 무기를 강화 할수 있는 시스템
    - 표창 키우기에서 유행하여 최근 방치형 게임중 캣 히어로, 아처 키우기등에서 볼수 있다. 

3) 요괴전기 타입
    - 중요 장비를 파밍하는 시스템
    - 요괴 전기의 경우 나무를 베면 무기가 파밍된다.
    - 그중 괜찮은 무기를 장착해서 강화하여 전투에 나가는 개념
    - 컨셉에 따라 무덤을 파밍하거나 대장간에서 무기를 담금질 하거나 하는 식으로 표현된다.

## CPI 테스트 2단계 리소스
