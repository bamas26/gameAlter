# 1943 The day after
## 1. 게임의 목적
- 1943년 노르망디 해변 근처에서 낙오된 연합국과 동맹군 병사들이
- 알수 없는 던전에서 만나 생존을 위해 함께 싸워가는 이야기
- 게임 요소
  - 골드로 기본 스탯 강화
  - 동료, 스킬, 보급(장비) 
## 2. 게임 시스템
- 기본 개념
  - 인간형 동료, 기계화 동료
    - 체력과 사정거리로 구분된다.
    - 기계화 동료의 경우 근거리 공격이 안되는 경우들이 있다.(ex 탱크)
  - 무게
    - 각각의 무게로 5등급이 있으며 등급차만큼 넉백이 이루어진다. (넉 시간동안 전투 제외)  
  - 높이
    - 5등급으로 높이가 구성 높이가 차이가 나면 명중이 낮아진다.
    - 인간의 경우 무기에 따라 높이가 추가 된다.
  - 컨디션
    - 계산된 모든 수치가 현재 동료의 컨디션에 따라 최종 적용된다.           
- 강화
  - 전투중 확보되는 골드로 스탯 강화
  - 1차 스탯 -> 순차적으로 활성화 되어 강화 할수 있다.
  - 2차 스탯 -> 마지막 오픈되는 일반 공격력 강화, 고급 공격력 강화 스탯 
- 스킬 (아래는 기본 보병 미군 스티브의 경우)
  - 기본 스킬 : 연사 -> 공속이후 2연사 혹은 3연사  
  - 고유 스킬 : 은폐 -> 쪼그려 쏴로 일정 시간동안 피격을 받지 않는다.  
  - 궁극 스킬 : 투척 -> 수류탄 투
- 동료
  - 스킬
    -   
  - 보병, 저격병, 오토바이병, 탱크병, 포격병, 탱크병등이 있다.
    - 톰은 탱크병.. 과 같은 형식으로 구성된다. 
  - 기본스킬 -> 연속 발사등 기본 스킬
  - 고유스킬 -> 25레벨에 도달하면 고유 스킬이 활성화 된다.
  - 궁극스킬 -> 동료의 인연이벤트를 클리어 하면 궁극 스킬을 사용할수 있다. 
- 보급
  - 음식(무기)과 물(방어구)이 기존 게임의 장비와 같은 개념으로 전제
  - 다만 둘다 소모되기 때문에 0이 되지 않게 해야 한다.
  - 약물은 만들어서 사용해야 한다.
    - 슬롯은 총 4개로 각각 음식,물,약물(치료),약물(영양)이다.

# 삼국무장소녀
## 1. 게임의 목적
- 삼국지 세계관의 무장들이 여성으로 등장
- 주장 세명중 한명을 선택하여 게임 진행
  - 주장 - 유비, 조조, 손권
- 무장 소녀들을 영입해 영토를 확장해 중국을 통일하는 게임
  - 지역의 무장 소녀들과 싸운다.
    - 원래 무장인 조운등 외에도 곽가류의 책사, 동탁류의 군장등도 등장
    - 무장의 구분 : 조운(전사), 황충(궁사), 곽가(법사), 동탁(군장)
      - 군장은 돈이 많기 때문에 스킬이 사병류이다.
        - 사병류 스킬
          - 기본 스킬이 호위 3인
          - 고유 스킬 : 기마대 등장
          - 궁극 스킬 : 무장 소녀를 휘하로 두어 무장소녀 장착이 사실상 늘어난다.  

## 2.1 게임 시스템
- 속성
  - 원작의 위,촉,오 국적을 속성으로 표현하여 각각 위류, 촉류, 오류로 부른다.
  - 속성은 컨디션에 영향을 미친다. 되도록이면 조조는 위류, 유비는 촉류의 무장을 사용하는게 좋다.
    - 위와 촉은 상극 -50%
    - 오는 25%
    - 정합일때 위촉오 각각 15%, 5%, 0% 
  - 무장소녀 이벤트를 통해 궁극의 스킬을 사용하게 되는 경우 속성이 변화한다.
    - 무장소녀 이벤트를 마무리 했을 경우 조조가 촉류의 무장 소녀를 무리 없이 사용할수 있다. 
- 유비,조조,손권중 하나를 선택하여 삼국 전쟁이 나선다.
- 동료
  - 팀으로 합류하여 협동전을 펼친다.
  - 다른 유저의 팀원중 한명이 합류 할수 있다.
  - 최초 31명 무장 소녀가 6티어로 구분되어 등장
    - 이후 1개월 단위로 1명의 무장 소녀 추가
    - 추가되는 무장소녀는 개발사의 기획에 따라 6티어중 하나로 추가 된다.  
- 장비
  - 주인공급인 조조, 유비, 손권의 장비로 무기와 방어구로 구분된다.
  - 반지와 목걸이는 던전에서 따로 제조 할수 있다.
- 스킬
  - 리더 스킬 :
    - 주인공이 사용하는 스킬로 주로 동료들의 능력을 강화하는 스킬이다.
    - 리더스킬을 사용하면 동료들의 리젠 시간이 무시된다. (리더가 콜하면 바로 스킬 사용 가능) 
    - 유비 : 단결 혹은 인화를 중심으로 한 스킬로 회복계 스킬이 많다.
    - 조조 : 공격을 중심으로 하는 스킬
    - 손권 : 수비를 중심으로 하는 스킬 
  - 개인 스킬
    - 각각의 무장소녀들의 특성에 따른 스킬로 각각 아래와 같다.
    - 소녀마다 기본 스킬 (두번베기 등과 같이 간단한...)
    - 고유 스킬 (25레벨에서 활성화 되는)
    - 궁극 스킬 : 무장소녀 이벤트를 클리어 하면 얻어지는 궁극의 스킬
  - 동료는 각각 기본 스킬과 25레렙로 성장하면 사용하는 동료 스킬이 있다.       
- 프로그레시브 뽑기
  - 동료, 장비, 스킬을 뽑는다.  
- 메인 메뉴
  - 강화, 주장, 동료, 던전, 훈련, 상점
  - 강화 : 골드로 강화 (공격력, 체력, 체력 회복, 치명 피해...) 
  - 주장 -> 조조, 유비, 손권을 강화
    - 장비, 스킬     
  - 동료 : 삼국지 인물 31명 장착및 보유 강화
  - 던전 : 보스(다이아), 산적토벌(골드) 마을 침략(장비), 마을 방어(명성), 무장소녀 이벤트(궁극 스킬), 공성 (반지 of 목걸이)
  - 훈련 : 개간(황무지를 개간하여 다이아등 재화를 얻는다.), 군사훈련(강화수치를 시간으로 % 강화 - 즉시 완료 다이아),
  - 상점 : 소환(동료,장비,스킬), 패키지, 무장소녀 구매, 재화 구매,

## 2.2 전투 시스템
- UI  
  - 주장 최초 등장 (항상 앞에 선다.) -> 캐릭터 모두 하단에 HP와 MP
  - 장착된 동료와 함께 등장
  - 마을 방어외엔 전진
  - 스킬 사용
    - 리더스킬 -> 중단부에 최대 6개까지 사용 가능
      - 리젠 타임이 지나면 사용 가능
      - ex) 유비 -> 전원 회복(체력 회복), 인화의 외침(컨디션 회복), 일제 사격 (투척계 모두 집중 난사)  
    - 개인 스킬
      - 각자 개인의 스킬을 리젠되면 사용 가능 (캐릭터 하단에 ) 

## 3. BM
- 소환
  - 프로그래시브 뽑기 (동료, 스킬, 장비)
  - 뽑기 레벨 존재 -> 레벨이 올라갈수록 확률 상승 -> 1일 3회 광고 보고 뽑기 가능 
- 패키지
  - 무장소녀를 중심으로 엮은 패키지
  - 무장소녀 + 던전 입장 열쇠 + 곡괭이 를 섞어서 만듬 
- 스페셜
  - 자동 축복, 광고 제거
  - 패쓰, 
- 재화
  - 다이아, 골드

# 몬스터 소녀전
## 1. 게임의 목적
- RPG 세계관의 몬스터들이 소녀로 등장 몬스터의 왕이 되기 위한 여정
  - 슬라임(소녀), 스켈레톤(슬랜더), 고렘(글래머)등 각각 몬스터의 특성을 여성을 풀어내어 캐릭터화 한다.
  - 몬스터 세계에서 여성으로 모예화 되지 않은 몬스터들을 상대하여 물리쳐야 한다.
- 게임 요소
  - 몬스터 소녀들을 영입하여 최강의 군대를 만들어 최대 6인이 싸우는 협동전
  - 지역의 몬스터 소녀들과 싸운다.
    - 몬스터 마다 원캐, 근캐로 나뉜다.

# 레전드오브피시
## 1. 게임의 목적
- 최고의 물고기가 되어
- 보물을 약팔하러 물고기의 바다를 침범하는 인간들을 물리치는 게임

## 2. 게임 시스템
- 기본 게임
  - 보물을 훔치러 바다를 침입한 인간들을 물고기 영웅이 물리친다.
  - 입에서 돌멩이를 물고 던진다. (등급에 따라 무기는 다양해 진다.)
  - 인간들을 도망치며 훔쳤던 골드를 다시 돌려준다.
 
- 모드 전환
  - 기본 모드는 보물 상자 바리게이트 뒤에서 안전하게 자동 공격하며 이를 방치 모드라 한다.
  - 방치시에 물고기를 업그레이드 시키며 자신감이 붙었다면 전투 버튼을 눌러 전투 모드에 돌입힌다.
  - 전투는 레벨로 구성되며 레벨마다 4개의 구간으로 구분되며 마지막 구간은 보스 구간이다.

- 물고기 영웅
  - 소환 : 골드를 지출하여 소환 (소환 등급은 최고등급 - 4등급) 8개를 소환 할수 있는 골드에 맞는 등급이 소환됨
    - 6개 연속 소환하면 광고 재 
  - 합성 : 같은 등급의 물고기를 합치면 상위 등급 물고기가 됨
  - 공급 : 다이아를 지불하고 물고기 공급 (공급 등급은 최고 등급 - 1등급)

# 고양이타일
## 1. 게임의 목적  
- 고양이가 물고기를 잡는 것 (타일을 클리어 하는것)
- 난이도가 상승되며 최고 스테이지인 199스테이지에 도달
  - 타일의 종류가 늘어난다.
  - 클리어할 블럭 갯수가 늘어난다. (수열로 늘어난다.)
  - 스테이지마다 터치 블럭 패턴이 달라진다.

## 2. 게임 설명
- 진입
  - 스테이지 진입 (이전 플레이한 스테이지에서 출발)
  - 방치 보상을 받으면 게임 시작 (광고 보고 2배)
- 플레이
  - 터치 블럭을 클릭하면 하단부 가이드 블럭으로 내려온다.
    - 같은 블럭이 있다면 연결, 합쳐서 3개가 되면 클리어
    - 레벨 게이지에 수치가 포함된다.
  - 승패
    - 다을레벨로 상승하면 골드로 보상 받는다.
      - 어느 레벨까지 가느냐가 목적
    - 가이드 블럭에 타일이 다 차면 패배
      - 10단위 스테이지로 이동
        - 36에서 실패 할경우 30부터 시작 (타일 배치가 동일하지 않다.)
        - 이어하려면 골드 지급 -> 돈이 없으면 광고를 봐야 한다.     
  - 아이템 플레이
    - 하단에 세가지 아이템을 사용 (골드로 차감)
      - 골드가 없으면 광고를 봐야 한다. 
    -  언도 -> 바로 전 플레이로 전환
    - 섞기 -> 블럭을 섞을수 있다.
    - 폭탄 -> 특정 블럭을 터트린다.(랜덤하게 생성하고 사용후 다시 재 생성)      
- 피버
  - 레벨 게이지 하단에 타임 게이지가 존재하며 풀로 차면 피버가 뜬다.
    - 밤위 되면서 피버 팡파레
    - 생선은 모두 초밥으로 변신 -> 클리어 하면 동전으로 보상 
  - 피버가 되면 생선은 모두 초밥으로 변한다.
    - 클리어 할때마다 돈으로 변한다 -> 골드가 느는게 목적 
  - 이때 터치 블록이 해금되어 전체를 다 터치 할수 있다.
    - 특정 시간이 지나면 다시 일반 판으로 이동한다.
